## 2. Bragg CDI SOM Pixel-Level Classification Demo

Files: `bragg_cdi_som_pixel_classification_demo.pptx` / `.pdf`

- SOM-based unsupervised, pixel-level classification of Bragg CDI patterns.
- Seven morphology prototypes (Core, Halo, Elongated, Streak, Diffuse, Ring, Tail) with class reuse across datasets.
- Demonstrates zero-label SOM segmentation, prototype transfer, and interpretable pattern taxonomy for coherent X-ray imaging.
